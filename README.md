A **desktop-based Crime Management System** built using **Python (Tkinter)** and **MySQL**, enabling efficient management of criminal records through a structured and user-friendly interface.

---

## 📌 Overview

This project is a **full-stack desktop application** that implements **CRUD operations**, integrates a **relational database**, and demonstrates **event-driven GUI architecture**.

It is designed to simulate real-world **law enforcement data systems**, focusing on performance, usability, and structured data handling.

---

## 🚀 Key Highlights 

- Built a **data-driven desktop application** using Python  
- Implemented **end-to-end CRUD operations** with MySQL  
- Designed a **responsive GUI** using Tkinter  
- Integrated **database connectivity using mysql-connector**  
- Developed **dynamic search functionality** with SQL queries  
- Applied **modular programming principles**  
- Demonstrates **software engineering fundamentals**:  
  - Data management  
  - UI/UX structuring  
  - Error handling  
  - State management  

---

## ✨ Features

- ➕ Add criminal records  
- 🔄 Update existing records  
- ❌ Delete records  
- 🔍 Search by Case ID, Criminal Number, or Name  
- 📋 View all records in a tabular format  
- 🧠 Structured storage with MySQL  
- 🖥️ Clean and interactive GUI
